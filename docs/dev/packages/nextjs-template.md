## Overview

This is the official Next.js template used for Dendron publishing (static site generation of your notes).

It powers the high-quality, fast publishing experience.

---

## Purpose

- Provide a ready-to-use Next.js setup optimized for Dendron content
- Handle sitemaps, SEO, and static export
- Serve as the target for `dendron publish` and related commands

---

## Key Features

- Next.js with TypeScript
- Custom sitemap generation
- Integration with Dendron's unified pipeline for content transformation
- Theming and component system
